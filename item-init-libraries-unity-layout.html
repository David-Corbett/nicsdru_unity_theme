4.2.1 init.libraries.unity-layout unity-layout($values)

Applies main layout widths based on the list of values passed in through a sass map variable.

Two values should be inserted for a 2 column layout and three values for a 3 column layout. The values should be declared from left to right as you expect to view the layout, e.g. $values = 19%, 60%, 19% - the first value will represent sidebar-first, the second value content and the third sidebar-second.

All values must be integers or rounded percentages. The sum of the values must not exceed 100.

Parameters:
  • $values
    width values
    Defaults to: 19%, 60%, 19%

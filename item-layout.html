2 layout Layout

Toggle example guides Toggle HTML markup

The theme provides a number of Drupal 'regions' namely:-

  • header: 'Site branding'
  • search: 'Search'
  • primary_menu: 'Main menu'
  • slideshow: 'Slideshow'
  • front_welcome: 'Front welcome'
  • top_first: 'First top'
  • top_second: 'Second top'
  • top_third: 'Third top'
  • breadcrumb: 'Breadcrumb'
  • sidebar_first: 'First sidebar'
  • sidebar_second: 'Sidebar second'
  • content: 'Content'
  • bottom_widget_1: 'Bottom widget 1'
  • bottom_widget_2: 'Bottom widget 2'
  • bottom_widget_3: 'Bottom widget 3'
  • footer_first: 'First Bottom'
  • footer_second: 'Second Bottom'
  • footer_third: 'Third Bottom'
  • footer_fourth: 'Fourth Bottom'
  • footer: 'Footer Bottom'

Code in the src/scss/layout directory of the theme primarily facilitates structuring and placement of these Drupal regions on a given page. Demonstration templates are provided below. Some regions are configured to only render on the front page of the site (these are outlined further in the documentation in the regions section of this page. Note that a Drupal region will only render if there is some code / markup in the region for Drupal to output. See here for more information on Drupal regions.

{# If there is no modifier set a default layout for a css grid so each grid item is 1 fr #}

  {% set modifier_class = 'no-sidebars' %}
{% endif %}
<div class="{{modifier_class}}{% if modifier_class == 'front' %} {{ 'two-sidebars' }}{% endif %}">
  <div id="header_wrapper">
    <div id="inner_header_wrapper">
      <header id="header">
        <div class="top_left">
          <div class="kss-layout-block">The site branding region.</div>
        </div>
        <div id="headerHero" class="top_right with-menu with-search">
          <div class="kss-layout-block">The search region.</div>
          <button class="burger-toggle collapse" type="button" data-toggle="collapse" data-target=".main-menu">
            <span class="sr-only">Menu</span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
          </button>
        </div>
        <div class="clear"></div>
      </header>
    </div>
    <div class="kss-layout-block">The main menu region.</div>
  </div>
  {% if modifier_class == 'front' %}
    <div class="kss-layout-block">The slideshow region.</div>
    <div class="kss-layout-block">The front welcome region.</div>
    <div id="top-area" class="clearfix">
      <div class="column kss-layout-block">The first top region.</div>
      <div class="column kss-layout-block">The second top region</div>
      <div class="column kss-layout-block">The third top region</div>
    </div>
  {% endif %}
  <div id="container">
    <div class="container-wrap">
      <div class="content-sidebar-wrap">
        <div id="content">
          <div class="kss-layout-block">The breadcrumb region</div>
          <main id="post-content">
            <div class="kss-layout-block">The content region</div>
          </main>
        </div>
        {% if modifier_class == 'front' or modifier_class == 'two-sidebars' or modifier_class == 'one-sidebar sidebar-first' %}
          <aside id="sidebar-first"><div class="kss-layout-block">The sidebar first region</div></aside>
        {% endif %}
      </div>
      {% if modifier_class == 'front' or modifier_class == 'two-sidebars' or modifier_class == 'one-sidebar sidebar-second' %}
        <aside id="sidebar-second"><div class="kss-layout-block">The sidebar second region</div></aside>
      {% endif %}
      {% if modifier_class == 'front' %}
        <div id="bottom_wrapper" class="footer_block bottom_widget">
          <div id="bottom-area" class="footer-area clearfix">
            <div class="column kss-layout-block">The bottom widget 1 region.</div>
            <div class="column kss-layout-block">The bottom widget 2 region.</div>
            <div class="column kss-layout-block">The bottom widget 3 region.</div>
          </div>
        </div>
      {% endif %}
    </div>
  </div>

        </div>

    </div>
  </div>
</div>

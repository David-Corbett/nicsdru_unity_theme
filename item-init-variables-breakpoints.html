4.1.2 init.variables.breakpoints Breakpoint variables

Set some values for where responsive changes occur - the values are set for use with min-width media queries.

Parameters:
  • $mob
    a mobile device view in landscape.
  • $phab
    a tablet sized device being viewed portrait;
  • $tab
    a tablet sized device being viewed landscape;
  • $lap
    a larger device with a smaller screen such as a laptop;
  • $desk
    a full sized screen, able to accommodate the largest sizes currently available;
  • $max-grid-width
    sets the maximum value the content area can expand to, on larger devices, the content container is centered on the page.
